# Task Manager CLI

A command-line task manager that saves data to JSON.

## How to Run

```bash
python task_manager.py
```

## Features

- Add tasks with custom status
- View all tasks numbered 1-N
- Update task status
- Delete tasks
- Data persists between sessions (saved to task.json)
